Welcome to My HTTP Server

My name is Basel Dawoud and I created this server in C.

Server Capabilities

This server can handle the following:

  • GET requests for static files (e.g., HTML).
  • Directory listing of available files in a requested directory.
  • Custom error messages for unsupported requests or missing resources.

Usage Instructions

You can interact with this server using a web browser or command-line tools like curl.

Here are some examples:

  • To retrieve the main page: curl http://localhost:8080/
  • To retrieve Hello file: curl http://localhost:8080/Hello
  • To retrieve another HTML page: curl http://localhost:8080/about.html
  • To list files in a directory: curl http://localhost:8080/test_directory/
  • To list a file in the test directory: curl http://localhost:8080/test_directory/test_file.html
  • To test a nonexistent resource: curl http://localhost:8080/nonexistent.html
